# AI Agent Mastery — 9-Week Curriculum Overview

## Learning Path

```
Prerequisites (10-15 hrs)
    │
    ▼
Week 1: Agent Fundamentals ──► Week 2: Framework Basics ──► Week 3: Basic Patterns
                                                                      │
Week 6: Framework Mastery ◄── Week 5: Context & Memory ◄── Week 4: Advanced Patterns
    │
    ▼
Week 7: MCP & A2A ──► Week 8: Safety & Security ──► Week 9: Testing & Deployment
```

## Ongoing Project: Topic Research Agent

The project evolves each week, building on previous work. Each week adds new capabilities while reinforcing earlier concepts. Both LangGraph and ADK implementations are maintained in parallel for comparison.

## Branch Strategy

Create a new branch for each week's work:
```bash
git checkout -b week-1-fundamentals
# ... do your work ...
git add -A && git commit -m "Complete Week 1: Basic agent with tracing"
git push origin week-1-fundamentals
```
